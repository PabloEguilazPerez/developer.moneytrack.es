# Webhooks

En esta guía, veremos cómo registrarse y consumir webhooks para integrar su aplicación con MoneyTrack. Con los webhooks, su aplicación puede saber cuándo sucede algo en el MoneyTrack, como que alguien borre una transaccion o agregue una categoria.{{ className: 'lead' }}

## Registrar webhooks

Para registrar un nuevo webhook, debe tener una URL en su aplicación a la que MoneyTrack pueda llamar. Puede configurar un nuevo webhook desde el panel de MoneyTracko en [Configuración de API](/login). Asigne un nombre a su webhook, elija los [eventos](#tipos de eventos) que desea escuchar y agregue su URL.

Ahora, cada vez que sucede algo de interés en su aplicación, el MoneyTrack activa un webhook. En la siguiente sección, veremos cómo consumir webhooks.

## Consumir webhooks

Cuando su aplicación reciba una solicitud de webhook del MoneyTrack, verifique el atributo `X-MoneyTrack-Event` para ver qué evento lo causó. La primera parte del tipo de evento le indicará el tipo de carga útil.

```json {{ title: 'Headers' }}
{
  "X-MoneyTrack-Event": "categoria.update",
  "X-MoneyTrack-Hook-ID": 1
}
```

```json {{ title: 'Ejemplo de carga útil de webhook' }}
{
  "nombre": "Viajes"
}
```
